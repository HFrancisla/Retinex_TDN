Retinex-TDN decomposition losses

代码来源：loss/decomposition_loss.py、loss/bsdp.py

PairedLoss：成对 low/high 分解损失

成对监督使用自重建、交叉重建和 R 一致性；逐像素 L 用于表达局部照明变化。

paired_pointL 为标量 [B,1,1,1]，用于全局光照分解。

损失名称默认权重数学公式该部分损失函数含义
high 自重建 recon_weight_high = 1.0 
L1(R_high ⊙ rep3(L_high), I_high)
 约束正常光样本由自身 R/L 重组回 I_high，保证分解不偏离原图，并固定 R 与 L 的基本尺度关系。
low 自重建 recon_weight_low = 0.3 
L1(R_low ⊙ rep3(L_low), I_low)
 约束低光样本由自身 R/L 重组回 I_low；权重较低，减少低光噪声和曝光不足对分解的过强牵引。
交叉重建 low cross_recon_weight_low = 0.001 
L1(R_high ⊙ rep3(L_low), I_low)
 用正常光反射配低光光照重建 I_low，检验 R 是否主要保存结构与颜色，而不是携带亮度。
交叉重建 high cross_recon_weight_high = 0.001 
L1(R_low ⊙ rep3(L_high), I_high)
 用低光反射配正常光光照重建 I_high，与上一项共同推动 R/L 解耦并可交换组合。
equal_R equal_r_weight = 0.1 
L1(R_low, sg(R_high))
 拉近同一场景 low/high 的反射图，使材质、纹理和边缘主要落在 R；sg 端作为停止梯度的目标。
符号注释：rep3(L)=cat(L,L,L)，表示把单通道光照复制成 3 通道；sg(.) 表示 detach()。

paired_pixelL 为逐像素图 [B,1,H,W]，用于建模 low/high 的空间光照分布。

损失名称默认权重数学公式该部分损失函数含义
high 自重建 recon_weight_high = 1.0 
L1(R_high ⊙ rep3(L_high), I_high)
 约束正常光样本由自身 R 和逐像素 L 重组回 I_high，保证局部光照图仍服务于原图重建。
low 自重建 recon_weight_low = 0.3 
L1(R_low ⊙ rep3(L_low), I_low)
 约束低光样本由自身 R/L 重组回 I_low；权重较低，减少低光噪声和曝光不足对分解的过强牵引。
交叉重建 low cross_recon_weight_low = 0.001 
L1(R_high ⊙ rep3(L_low), I_low)
 用正常光反射配低光光照重建 I_low，检验 R 是否主要保存结构与颜色，而不是携带亮度。
交叉重建 high cross_recon_weight_high = 0.001 
L1(R_low ⊙ rep3(L_high), I_high)
 用低光反射配正常光光照重建 I_high，与上一项共同推动 R/L 解耦并可交换组合。
equal_R equal_r_weight = 0.1 
L1(R_low, sg(R_high))
 拉近同一场景 low/high 的反射图，使材质、纹理和边缘主要落在 R；sg 端作为停止梯度的目标。
Retinex smooth smooth_weight = 0.1 
Smooth(L_low, R_low) + Smooth(L_high, R_high)
 惩罚 L 的空间起伏，并在 R 的强边缘处降低惩罚；让光照缓慢变化，纹理和细节留给 R。
符号注释：rep3(L)=cat(L,L,L)，表示把单通道光照复制成 3 通道；sg(.) 表示 detach()。

UnpairedLoss：非配对 low/high 分解损失

没有同场景配对关系，所以不使用交叉重建和 equal_R；改用重建、anchor、BDSP 和 self recon。

unpaired_pointL 为标量，anchor 约束 L 接近 max_c(I)。

损失名称默认权重数学公式该部分损失函数含义
low/high 重建 recon_weight = 1 
L1(R_low ⊙ rep3(L_low), I_low)
+ L1(R_high ⊙ rep3(L_high), I_high)
 分别用各自 R/L 重组 low 与 high 输入，提供非配对训练的基础自监督信号，防止分解结果脱离原图。
AnchorPoint anchor_weight = 0.05 
L1(max_c(I_low), expand(L_low))
+ L1(max_c(I_high), expand(L_high))
 用输入亮度最大通道给标量 L 定尺度，避免 L 过暗或过亮后由 R 被动补偿。
BDSP 结构保持 bdsp_weight = 0.05 
L1(BDSP_Face(R_low), BDSP_Face(I_low))
+ L1(BDSP_Face(R_high), BDSP_Face(I_high))
 让 R 的边缘响应贴近输入图，保留轮廓和局部结构，避免反射分量过度平滑或丢边。
Self recon self_recon_weight = 0.05 
L1(L_R_low, L_R_high)
 将反射图再次分解得到二次 L，并约束 low/high 的二次光照一致，减少非配对样本间的分解漂移。
符号注释：rep3(L)=cat(L,L,L)，表示把单通道光照复制成 3 通道；max_c(I) 表示 RGB 通道最大值图。

unpaired_pixelL 为逐像素图，anchor 只约束平均亮度，避免逐像素亮度拷贝。

损失名称默认权重数学公式该部分损失函数含义
low/high 重建 recon_weight = 1 
L1(R_low ⊙ rep3(L_low), I_low)
+ L1(R_high ⊙ rep3(L_high), I_high)
 分别用各自 R/L 重组 low 与 high 输入，提供非配对训练的基础自监督信号，防止分解结果脱离原图。
AnchorPixel anchor_weight = 0.05 
L1(mean_hw(max_c(I_low)), mean_hw(L_low))
+ L1(mean_hw(max_c(I_high)), mean_hw(L_high))
 只约束 L 的全图平均亮度，固定整体曝光尺度，同时保留局部光照变化自由度。
BDSP 结构保持 bdsp_weight = 0.05 
L1(BDSP_Face(R_low), BDSP_Face(I_low))
+ L1(BDSP_Face(R_high), BDSP_Face(I_high))
 让 R 的边缘响应贴近输入图，保留轮廓和局部结构，避免反射分量过度平滑或丢边。
Self recon self_recon_weight = 0.05 
L1(L_R_low, L_R_high)
 将反射图再次分解得到二次 L，并约束 low/high 的二次光照一致，减少非配对样本间的分解漂移。
Retinex smooth smooth_weight = 0 
Smooth(L_low, R_low) + Smooth(L_high, R_high)
 抑制逐像素 L 的高频噪声，使光照图表达平滑照明而不是纹理细节。
符号注释：rep3(L)=cat(L,L,L)，表示把单通道光照复制成 3 通道；max_c(I) 表示 RGB 通道最大值图；mean_hw(.) 表示空间均值。

PureLowDoubleLoss：仅 low 双视图自监督损失

一张低光图构造两个增强 view；该类独有 reflect 项，用双向 detach 约束两个 view 的 R 一致。

pure_low_double_point两个 view 分别输出标量 L，用于双视图全局光照分解。

损失名称默认权重数学公式该部分损失函数含义
双视图重建 recon_weight = 1.0 
L1(R1 ⊙ rep3(L1), I1)
+ L1(R2 ⊙ rep3(L2), I2)
 两个增强 view 都要由各自 R/L 重组回输入，确保增强后的视图仍能被同一分解框架解释。
AnchorPoint anchor_weight = 0.05 
L1(max_c(I1), expand(L1))
+ L1(max_c(I2), expand(L2))
 给两个 view 的标量 L 分别定亮度尺度，避免双视图训练中 R/L 比例任意漂移。
BDSP 结构保持 bdsp_weight = 0.05 
L1(BDSP_Face(R1), BDSP_Face(I1))
+ L1(BDSP_Face(R2), BDSP_Face(I2))
 让两个 view 的 R 都保留各自输入的边缘结构，使反射分量承载稳定内容而非仅拟合亮度。
Self recon self_recon_weight = 0.05 
L1(LR1, LR2)
 对两个 view 的 R 再分解并对齐二次 L，要求同一内容在增强前后得到相近的光照解释。
Reflect reflect_weight = 0.0 
L1(R1, sg(R2)) + L1(R2, sg(R1))
 直接拉近两个 view 的 R，强化同一低光内容的反射一致性；双向 sg 让每次只更新预测侧。
符号注释：rep3(L)=cat(L,L,L)，表示把单通道光照复制成 3 通道；max_c(I) 表示 RGB 通道最大值图；sg(.) 表示 detach()。

pure_low_double_pixel两个 view 分别输出逐像素 L，用于对齐双视图的局部光照解释。

损失名称默认权重数学公式该部分损失函数含义
双视图重建 recon_weight = 1.0 
L1(R1 ⊙ rep3(L1), I1)
+ L1(R2 ⊙ rep3(L2), I2)
 两个增强 view 都要由各自 R/L 重组回输入，确保逐像素光照图仍围绕原图解释，而不是自由漂移。
AnchorPixel anchor_weight = 0.05 
L1(mean_hw(max_c(I1)), mean_hw(L1))
+ L1(mean_hw(max_c(I2)), mean_hw(L2))
 只锚定每个 view 的平均亮度，保留局部照明差异，同时降低 L 直接复制纹理的风险。
BDSP 结构保持 bdsp_weight = 0.05 
L1(BDSP_Face(R1), BDSP_Face(I1))
+ L1(BDSP_Face(R2), BDSP_Face(I2))
 让两个 view 的 R 都保留各自输入的边缘结构，使反射分量承载稳定内容而非仅拟合亮度。
Self recon self_recon_weight = 0.05 
L1(LR1, LR2)
 对两个 view 的 R 再分解并对齐二次 L，要求同一内容在增强前后得到相近的光照解释。
Reflect reflect_weight = 0.0 
L1(R1, sg(R2)) + L1(R2, sg(R1))
 直接拉近两个 view 的 R，强化同一低光内容的反射一致性；双向 sg 让每次只更新预测侧。
Retinex smooth smooth_weight = 0 
Smooth(L1, R1) + Smooth(L2, R2)
 抑制两个 L 图中的高频抖动，并在 R 的边缘处保留合理跳变，使局部照明更稳定。
符号注释：rep3(L)=cat(L,L,L)，表示把单通道光照复制成 3 通道；max_c(I) 表示 RGB 通道最大值图；mean_hw(.) 表示空间均值；sg(.) 表示 detach()。

PureLowSingleLoss：仅 low 单视图自监督损失

最轻量的 low-only 训练目标：重建、anchor、BDSP；逐像素 L 用于表达单图局部照明。

pure_low_single_point单张 low 图输出标量 L，用于单视图全局光照分解。

损失名称默认权重数学公式该部分损失函数含义
单图重建 recon_weight = 1.0 
L1(R ⊙ rep3(L), I)
 要求单张 low 图能由自身 R/L 重组回原图，提供单视图训练的核心自监督信号。
AnchorPoint anchor_weight = 0.05 
L1(max_c(I), expand(L))
 用输入亮度给全局 L 定尺度，避免标量光照任意漂移，并稳定 R 的亮度补偿范围。
BDSP 结构保持 bdsp_weight = 0.05 
L1(BDSP_Face(R), BDSP_Face(I))
 让 R 保留输入的主要轮廓、纹理和边缘，弥补单视图模式缺少跨图一致性约束的问题。
符号注释：rep3(L)=cat(L,L,L)，表示把单通道光照复制成 3 通道；max_c(I) 表示 RGB 通道最大值图。

pure_low_single_pixel单张 low 图输出逐像素 L，用于建模局部亮度起伏。

损失名称默认权重数学公式该部分损失函数含义
单图重建 recon_weight = 1.0 
L1(R ⊙ rep3(L), I)
 要求单张 low 图能由自身 R/L 重组回原图，保证逐像素光照图仍服务于图像重建。
AnchorPixel anchor_weight = 0.05 
L1(mean_hw(max_c(I)), mean_hw(L))
 只锚定 L 的全图平均亮度，固定整体曝光尺度，同时允许 L 表达局部照明起伏。
BDSP 结构保持 bdsp_weight = 0.05 
L1(BDSP_Face(R), BDSP_Face(I))
 让 R 保留输入的主要轮廓、纹理和边缘，避免逐像素 L 把结构细节吸走。
Retinex smooth smooth_weight = 0 
Smooth(L, R)
 惩罚 L 的局部高频变化，使光照图更像缓慢变化的照明场，减少纹理泄漏到 L。
符号注释：rep3(L)=cat(L,L,L)，表示把单通道光照复制成 3 通道；max_c(I) 表示 RGB 通道最大值图；mean_hw(.) 表示空间均值。
